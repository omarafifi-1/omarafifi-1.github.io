## Portfolio Design

This portfolio website has a clean, professional appearance with:

- **Responsive Layout**: Automatically resizes to all screen sizes from mobile to desktop
- **Navigation**: Fixed-top navbar that alters transparency on scroll
- **Hero Section**: Full-width banner with stylish text.
- **Card-Based Projects**: Visual project shows with hover effects.
- **Skill Visualization**: Skill Visualization: Progress bars showing technical levels of proficiency.
- **Contact Form**: Interactive form with validation.

### Design Features
- **Color Scheme**: Dark/light professional contrast with accent colors
- **Typography**: Clean, readable fonts with correct hierarchy
- **Animations**: Subtle scroll reveals and hover effects
- **Accessibility**: WCAG-accessible design features
- **Performance**: Compressed images and minimized HTTP requests

## Technical Implementation
- Bootstrap 5 grid system for responsive design
- Custom CSS variables for consistent styling
- JavaScript for interactive components and functions like altering transparency of navbar
